telegraf

Telegraf 源码阅读笔记(1)

背景

telegraf 是 influxdata 公司开源的一款插件化的，用来做指标收集、变换、聚合、输出的时序数据收集器。
influxdata 公司本身是做物联网数据存储、分析的公司，核心资产是 influxdb 时序数据库，其核心解决方案是 TICK 套装(Telegraf + Influxdb + Chronograf + Kapacitor)。

约束

写本文时, telegraf 最新版本 = v1.14
